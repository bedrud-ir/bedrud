# Bedrud Documentation

Welcome to the Bedrud project documentation. This is a platform for video meetings and collaboration.

## Getting Started
If you are a developer, start here:
*   [Makefile Guide](./makefile.md) - Learn how to run and build the project.

## Architecture
Learn how the different parts of the project work:

*   [Codebase Structure](./codebase_structure.md) - Detailed technical guide for developers.

## Core Features
- **Video Meetings:** High-quality video and audio using LiveKit.
- **Secure Login:** Support for email/password and social login (Google/GitHub).
- **Room Management:** Easy to create and join private or public rooms.
- **Admin Dashboard:** Tools for administrators to manage users and rooms.
